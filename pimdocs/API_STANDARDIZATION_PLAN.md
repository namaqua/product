## üéØ Standardization Rules

### Collection Responses (Lists/Arrays)
```json
{
  "success": true,
  "data": {
    "items": [...],
    "meta": {
      "totalItems": 100,
      "itemCount": 20,
      "page": 1,
      "totalPages": 5,
      "itemsPerPage": 20,
      "hasNext": true,
      "hasPrevious": false
    }
  },
  "timestamp": "2025-01-09T..."
}
```

### Action Responses (Create/Update/Delete)
```json
{
  "success": true,
  "data": {
    "item": {...},
    "message": "Created successfully"
  },
  "timestamp": "2025-01-09T..."
}
```

### Single Item Responses (Get by ID)
```json
{
  "success": true,
  "data": {...},
  "timestamp": "2025-01-09T..."
}
```
